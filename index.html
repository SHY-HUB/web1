WEB 이란 무엇인가?

Introduced in 1990, the World Wide Web ( WWW , W3) is a term used to refer to spaces, such as webs, that can be used to share information with people using computers connected to the Internet . Hypertext pages written in HTML based on the HTTP protocol are configured to be read by a specific program called a web browser . Also known as WWW , W3, or simply Web. Since 1994, when the Internet began to be distributed to the general public, it is the most widely used Internet system to be treated as synonymous with the Internet. If you ask the public what it's like to be on the Internet, most will answer that they are turning on their web browser and surfing the web. Since its release, it has been continuously improving, adding features of HTML itself, supporting better design with CSS, and creating dynamic web pages instead of static web pages with JavaScript and enabling full-scale programming. , And many other JavaScript-based libraries, including jQuery , PHP , ASP With the advent of various server-side frameworks, such as , JSP , CGI , Node.js , Django , Ruby on Rails, etc., as of 2015, it has enjoyed its position as another software platform, absorbing almost all other Internet system features. As of 2017, the latest standards are HTML5 , CSS 3, ECMAScript 7. You can check the acid test for compliance with web browser standards .
